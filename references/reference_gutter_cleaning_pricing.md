## Gutter Cleaning — Sqft Tier Pricing (Good Package Base)

| Sqft | Good (Clean Only) |
|---|---|
| Up to 1,500 sqft | $199 |
| 1,500–2,000 sqft | $249 |
| 2,000–2,500 sqft | $299 |
| 2,500–3,000 sqft | $349 |
| 3,000–3,500 sqft | $399 |
| 3,500+ sqft | $449 |

## Package Upgrades
- **Better** = Good base + Better service (flush downspouts, snake drains)
  - Pricing: Good base + ~$50-75 depending on job
- **Best** = Better + full service

## Caulking Add-Ons (Flat Fee)
- **Partial/spot caulking** (1-story, Better package): +$150
- **Full caulk** (Best package — takes a couple hours): +$250 minimum

## Example: 2,684 sqft (Sons Holdings)
- Good: $349 (2,500–3,000 tier)
- Better + partial caulk: $349 + $150 = **$499**
- Best + full caulk: $349 + $250 = **$599**
