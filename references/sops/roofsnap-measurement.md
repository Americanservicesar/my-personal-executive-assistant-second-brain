# RoofSnap Measurement SOP

> Standard operating procedure for requesting and creating roof/gutter measurements
> via RoofSnap web portal. Used by Milli (sales) and Buddy (bids/partnerships).

---

## When to Use RoofSnap

- Gutter installation quotes (need exact linear feet)
- Roof-related proposals (need exact square footage)
- Apex Shield Construction jobs
- Any proposal where accurate roof/gutter measurements improve pricing confidence

---

## Trigger

When Anthony says: "Measure [address] for gutters" or "Get measurements for [property]"

---

## Routing

1. **Vizzy** receives the request
2. Routes to:
   - **Milli** — if it's a direct sales lead
   - **Buddy** — if it's a bid/partnership opportunity
3. Assigned agent executes the measurement

---

## Measurement Workflow

### Step 1 — Log into RoofSnap
- Go to RoofSnap web portal (roofsnap.com)
- Log in with Anthony's credentials
- Navigate to "New Project"

### Step 2 — Create Measurement
1. Enter the property address
2. RoofSnap pulls satellite imagery
3. Trace roof edges / gutter lines on the image
4. RoofSnap calculates:
   - Total roof square footage
   - Individual roof face areas
   - Gutter linear footage (per run)
   - Ridge, hip, valley lengths
   - Pitch estimates

### Step 3 — Export Data
- Save the project in RoofSnap
- Export measurement report (PDF or data)
- Note key measurements:
  - Total gutter linear feet
  - Total roof square feet
  - Number of downspouts needed
  - Any special conditions (valleys, dormers, multi-story)

### Step 4 — Log to Memory
Update `memory/pricing-actuals.md` → RoofSnap Measurement Log:

| Date | Address | Roof Sq Ft | Gutter Lin Ft | Agent | Proposal # | Notes |
|------|---------|-----------|---------------|-------|-----------|-------|

### Step 5 — Feed into Proposal Builder
1. Open proposal-builder skill
2. Input actual measurements from RoofSnap
3. Pricing confidence upgrades from Medium/Low to High
4. Generate accurate proposal with real measurements

---

## Tips
- RoofSnap measurements are estimates based on satellite imagery — verify on-site for large jobs
- For multi-building properties (apartments), measure each building separately
- Save all projects in RoofSnap for future reference
- If the satellite image is outdated or unclear, note it in the proposal as a caveat
