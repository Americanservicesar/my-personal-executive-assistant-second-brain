## Connected to Seomi Already
- WordPress REST API (americanservicesar.com) ✅
- RankMath PRO API ✅
- PageSpeed Insights API (free) ✅
- SerpApi ✅
- Web Search (Tavily) ✅

## FREE Tools to Connect Next
1. Google Search Console API — n8n has built-in node, needs Google OAuth
2. Google Analytics 4 API — n8n has built-in node, needs Google OAuth
3. Bing Webmaster Tools API — free, HTTP node with API key
4. Moz API free tier — 50 requests/month for Domain Authority checks
5. Google Business Profile API — free, HTTP + OAuth2

## Installed on WordPress (manual use, no API for n8n)
- Search Atlas SEO plugin — no API
- Ahrefs SEO plugin — no API from n8n (but installed on site)

## NOT worth buying for automation
- SEMrush ($140+/mo) — no budget
- Ahrefs ($129+/mo) — no budget
- Surfer SEO ($89+/mo) — no real API
- HARPA AI — browser extension only, no API
- Search Atlas paid ($99+/mo) — no API

## Budget recommendation: $0-5/month
Google free tools cover 90%. Copyscape at $0.03/check if needed.
