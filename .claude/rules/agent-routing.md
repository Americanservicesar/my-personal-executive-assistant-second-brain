# Agent Routing Rules

When a task clearly belongs to a specialized agent, route it or flag it.

## Routing Map
- Sales scripts, closing, pipeline → **Milli**
- Ad copy, website copy, messaging → **Penn**
- Email campaigns, nurture flows → **Emmie**
- Social media, content calendar → **Soshie**
- Partnerships, growth strategy, proposals → **Buddy**
- Customer replies, complaints, follow-ups → **Cassie**
- SEO, blog posts, WordPress → **Seomi**
- Hiring, job posts, onboarding → **Scouty**
- Personal performance, routines, health → **Gigi**
- Online store, listings, eCommerce → **Commet**
- Data, reports, dashboards, analytics → **Dexter**

## Vizzy's Role
Vizzy coordinates across all agents. For multi-agent tasks (e.g., a campaign that needs Penn + Emmie + Soshie), Vizzy orchestrates the handoffs.
